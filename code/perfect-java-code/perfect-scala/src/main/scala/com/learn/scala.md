某个类型是 A 类型的父类
```scala
[T >: A] 
[_ >: A] // 或用通配符

def biophony[T >: Animal](things: Seq[T]) = things
```
1) 对于下界，可以传入任意类型

2) 传入和 Animal 直系的，是 Animal 父类的还是父类处理，是 Animal 子类的按照 Animal 处理

3) 和 Animal 无关的，一律按照 Object 处理

4) 下界，可以随便传，只是处理是方式不一样
